**场景**：
长列表（FlatList）数据更新或滚动时出现明显卡顿或白屏。

**优化方案**：
- **为 Item 增加 Key**：在 `map` 循环或列表渲染中，给每个 Item 设置唯一的 `key`。
  - *作用*：帮助 React 识别哪些 Item 是新增、删除或移动的，仅复用或更新变化的 Item，而非销毁重建整个列表。
- **开启子视图剪裁 (removeClippedSubviews)**：设置 `removeClippedSubviews={true}`。
  - *作用*：将屏幕可视区域之外的视图从原生视图树中移除（"下树"），释放原生内存和 GPU 资源。

**收益**：
- **内存优化**：减少不可见视图的资源占用。
- **流畅滚动**：减少滚动时的渲染压力，保持高 FPS。
